Regra da Cascata

O CSS funciona em cascata, ou seja quando há conflito, a última regra é a que vale.

Especificidade

A ordem de força dos seletores é:

  • ID (#id) é o mais forte
  • Classe (.classe) é médio
  • Tag (div, h1, p) é o mais fraco

Combinação de Seletores

.primeiro h1 - seleciona h1 dentro de .primeiro

.primeira > h1 - seleciona h1 filho direto de .primeira

.primeira h1 + h1 - seleciona o h1 logo após outro h1

Pseudo-classes

  • :first-child - primeiro filho
  • :last-child - último filho
  • :nth-child(n) - filho na posição n
  • :nth-of-type(n) - filho de um tipo específico

Display Inline e Block

Display block ocupa toda a linha. Exemplos: div, p, h1.

Display inline ocupa só o espaço do conteúdo. Exemplos: span, a, strong.

Box Model

O box model tem margin, padding e border.

Shorthand:

  • 1 valor: margin: 10px
  • 2 valores: margin: 10px 20px
  • 3 valores: margin: 10px 20px 30px
  • 4 valores: margin: 10px 20px 30px 40px

Com box-sizing: border-box a largura já tem padding e border.
